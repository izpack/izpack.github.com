+UserPathPanel
+--------------
+
+This panel allows the user to select a path similar to the installation
path.
+It can be pre-loaded with a path and set to only display if a certain pack
is
+selected using the following variables:
+
+    <variable name="UserPathPanelVariable" value="@{default.dest.dir.sql
}"/>
+    <variable name="UserPathPanelDependsName" value="Install Database
Server"/>
+
+Final path selection can be accessed using the variable
"UserPathPanelVariable".
